# Gemini Timeline (Gemini 时间轴)

一个极简、优雅的 Google Gemini 侧边栏对话导航插件。
采用透明悬浮设计，支持 3D 滚轮浏览效果，让历史记录管理变得充满科技感。

![Version](https://img.shields.io/badge/version-1.0-blue)

## ✨ 功能特点

- **极致极简**: 默认透明悬浮，无干扰，完美融入界面。
- **3D 滚轮**: 独特的滚动遮罩效果，像拨动时间轮盘。
- **自动定位**: 鼠标悬停自动吸附到当前对话位置。
- **智能高亮**: 当前阅读的对话会高亮显示，悬停发光。
- **隐私安全**: 代码开源，所有数据仅存储在本地浏览器。

## 📥 如何安装 (开发者模式)

由于本项目尚未上架 Chrome 商店，您需要通过“开发者模式”安装：

1. **下载代码**: 点击本页面右上角的绿色 **Code** 按钮 -> **Download ZIP**。
2. **解压文件**: 将下载的压缩包解压到一个文件夹中。
3. **打开扩展管理**: 在 Chrome/Edge 浏览器地址栏输入 `chrome://extensions/` 并回车。
4. **开启开发者模式**: 打开右上角的开关 **Developer mode**。
5. **加载**: 点击左上角的 **Load unpacked** (加载已解压的扩展程序)。
6. **选择文件夹**: 选择你刚才解压的文件夹即可。
